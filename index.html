Setup

Distributed as a Docker image. Running in under 2 minutes.

mayurG2299/s3-portal:latest docker-compose.yml

Quick Start

# Copy the compose file
curl -LO https://github.com/mayurG2299/s3-portal/raw/main/docker-compose.yml

# Configure environment
cp .env.example .env
nano .env

# Start
docker compose up -d

Visit http://localhost:3000

First time only - run migrations:

Migrations

docker compose run --rm app npx prisma migrate deploy

Features

File management

Upload, download, preview and tag files across S3 buckets.

Shareable links

Secure expirable share links without exposing S3 directly.

Team access control

Role-based permissions: Owner, Admin, and Viewer.

Multiple credentials

Connect multiple AWS accounts from one dashboard.

Audit logs

Full activity log for all file operations and team actions.

Self-hosted

Your data stays on your infrastructure, no third-party services.
